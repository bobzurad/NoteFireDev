NoteFire

NoteFire is a collection of simple note apps. Each app is functionally identical, but written using a different Firebase JavaScript Library. The goal of this project is to show how a simple notes app can be written using the libraries that Firebase offers. Each app points to the same Firebase, so each app contains identical data. The source code to these apps are availabe on GitHub.
